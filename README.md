#### Node Config.
- Listando as versões instaladas do Node na minha máquina:
```bash
nvm ls
```

- Remoção da versão atual do node:
```bash
nvm uninstall vxx.xx.xx
```

- Instalação do Node:
```bash
nvm install v22.17.1
```

- Editado o path para a versão correta do node:
```bash
sudo nano ~/.bashrc
```
